## 1. Цель и аудитория

### Цель сайта

**Главная:** Генерировать входящие заявки от владельцев бизнеса на разработку сайтов, приложений и AI-агентов.

**Вторичные:**
- Демонстрировать компетентность через портфолио и кейсы
- Позиционировать специалиста как эксперта, а не просто исполнителя
- Отсекать нецелевые заявки (слишком маленький бюджет, нереальные ожидания)

**Целевое действие:** Запись на бесплатный 30-минутный созвон (Calendly) **или** заполнение формы "Обсудить проект".

**KPI:**
- Конверсия посетитель → заявка: **цель 4–6%** (рынок: 1–3%)
- PageSpeed Score: **≥ 90** (мобайл и десктоп)
- Bounce Rate: **< 60%**

---

### Целевая аудитория

**Первичная (основной покупатель):**
- Владелец малого/среднего бизнеса, 30–50 лет
- Есть конкретная задача: нужен сайт, приложение или автоматизация
- Не технарь — не понимает терминологию, думает категориями результата и денег
- Уже обжигался на некачественной разработке или затянутых сроках
- Принимает решения быстро, но хочет чувствовать безопасность сделки

**Вторичная (реферальный источник):**
- Маркетологи и менеджеры, которые ищут подрядчика для своего клиента
- Другие разработчики, которые хотят делегировать AI-часть

**Боли аудитории:**
1. "Нет технического партнёра, которому можно доверять"
2. "Прошлый разработчик сдал не то, не вовремя или пропал"
3. "Хочу AI-автоматизацию, но не знаю, с чего начать"
4. "Сайт устарел и не приносит заявки"
5. "Теряю часы на задачи, которые мог бы делать алгоритм"

## 2. Оффер и позиционирование

### Главный оффер

> **"Строю сайты, приложения и AI-агентов для бизнеса — с фиксированной ценой, в срок, с гарантией результата"**

### Уникальное отличие (USP)

AI-агенты как дифференциатор — большинство разработчиков предлагают только сайты и приложения. Связка "сайт + приложение + AI-автоматизация" под одной рукой — редкость, это позиционирует как цифрового партнёра, а не просто кодера.

### Варианты заголовка для A/B тестирования

| Вариант | Формула | Текст |
|---------|---------|-------|
| A | Outcome + Audience | "Цифровые продукты для бизнеса, которые работают на тебя" |
| B | I Build / That Does | "Строю сайты, приложения и AI-агентов, которые растят твой бизнес" |
| C | Bold Claim | "Разработка без кота в мешке — фиксированная цена, фиксированный срок" |

**Рекомендуется начать с варианта A** — наиболее универсальный.

### СЕКЦИЯ 09 — FAQ

**Цель:** Закрыть оставшиеся возражения прямо на странице.

**Заголовок:** "Частые вопросы"

**Аккордеон с 6 вопросами:**

1. **"Я не знаю точно, что мне нужно"**  
   → Отлично — именно для этого первый созвон бесплатный. Разберёмся вместе: я задам правильные вопросы и предложу оптимальное решение под твою задачу.

2. **"Сколько времени займёт проект?"**  
   → Лендинг — 2–3 недели, приложение — от 4 недель, AI-агент — от 2 недель. Конкретные сроки фиксируются в предложении до старта — и я их держу.

3. **"Я уже обжигался на разработчиках"**  
   → Понимаю. Именно поэтому работаю с зафиксированным объёмом и ценой в договоре, промежуточными демо каждые 1–2 недели и честным общением, если что-то меняется.

4. **"Что будет после запуска?"**  
   → Первые 30 дней — бесплатные правки и поддержка. Далее — по договорённости: разовые задачи или ежемесячный retainer.

5. **"Ты работаешь один или у тебя команда?"**  
   → Работаю лично — ты получаешь прямой контакт с исполнителем без посредников. На крупных проектах подключаю проверенных партнёров, о чём предупреждаю заранее.

6. **"Что такое AI-агент и зачем он моему бизнесу?"**  
   → AI-агент — это программа, которая выполняет задачи вместо человека: отвечает на вопросы клиентов, обрабатывает заявки, анализирует данные. Если ты тратишь время на повторяющиеся задачи — скорее всего, AI может их автоматизировать.

## 7. Анимации и интерактивность

### Анимации по секциям

| Секция | Анимация | Библиотека | Описание |
|--------|----------|-----------|----------|
| **Hero** | Background effect | Vanta.js NET | Сетка частиц, реагирует на мышь |
| **Hero** | H1 typewriter | typed.js | "сайты / приложения / AI-агентов" |
| **Hero** | Stats counter | Custom JS | Счётчик до цифры при появлении |
| **Все секции** | Fade in up | Framer Motion | Появление при скролле |
| **Карточки** | Scale on hover | CSS transition | transform: scale(1.02) |
| **CTA-кнопка** | Glow pulse | CSS animation | Мигающий glow эффект |
| **Навигация** | Blur backdrop | CSS | На скролле появляется `backdrop-filter` |
| **Cursor** | Custom cursor | GSAP | Кастомный курсор, magnetic на кнопках |

### Typewriter эффект в Hero

```jsx
// Подзаголовок меняет специализацию
"Строю [сайты → приложения → AI-агентов] для бизнеса"

// Реализация через typed.js:
new Typed('#hero-typed', {
  strings: ['сайты', 'приложения', 'AI-агентов'],
  typeSpeed: 70,
  backSpeed: 40,
  backDelay: 2000,
  loop: true,
});
```

### Scroll-анимации (Framer Motion)

```tsx
// Единый компонент для появления секций
const FadeInSection = ({ children, delay = 0 }) => (
  <motion.div
    initial={{ opacity: 0, y: 40 }}
    whileInView={{ opacity: 1, y: 0 }}
    transition={{ duration: 0.6, delay, ease: 'easeOut' }}
    viewport={{ once: true, margin: '-100px' }}
  >
    {children}
  </motion.div>
);
```

### Animated Counter (Stats bar)

```tsx
// Счётчик анимируется при попадании в viewport
// 0 → 47 (проектов), 0 → 5 (лет), etc.
// Используем useInView + requestAnimationFrame
```

### Правила анимаций

- Каждая анимация должна нести **смысл** (привлечь внимание, направить, подтвердить)
- `prefers-reduced-motion` — отключать все анимации для пользователей с таким предпочтением
- Анимации появления — только `once: true`, не при каждом скролле
- Продолжительность: 300–600ms, easing: `easeOut`
- Не анимировать одновременно более 3 элементов на экране

## 8. Формы и конверсионные элементы

### Контактная форма

**Поля (максимум 4):**
1. Имя `[text, required]`
2. Email или Telegram `[email/text, required]`
3. Тип проекта `[select: Сайт / Приложение / AI-агент / Не знаю]`
4. Коротко о задаче `[textarea, optional, placeholder: "Например: нужен чатбот для обработки входящих заявок"]`

**Кнопка:** "Отправить заявку →"

**После отправки:**
- Спиннер на кнопке пока идёт запрос
- Success state: "Получил! Отвечу в течение 24 часов 🎉"
- Error state: "Что-то пошло не так. Напиши мне напрямую: [email]"

**Обработка:** Resend API или EmailJS (без бэкенда)
